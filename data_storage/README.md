# データ定義
JSONデータの定義を以下に示す。データは2025年シーズンのものである。

## 基本項目
- team: 所属チーム
- position: 登録ポジション
- number: 背番号
- nameJ: 氏名（日本語）

## 打撃項目
- maxEV: 最高打球速度
- maxDistance: 最長飛距離
- maxBatspeed: 最高スイング速度
- HardHit%: ハードヒット率
- Barrel%: バレル率
- Chase%: ボールゾーンスイング率

## スプリント項目
- sprint: スプリント速度
- runtime1B: 最速タイム（一塁到達）

## 守備項目
- maxThrowVelo: 最高送球速度

## 追加項目
- nameE: 氏名（英語）
- fgId: FanGraphsの選手ID
